# 🗳️ Democracy Guide – Understanding Elections

An interactive and beginner-friendly website that explains the complete election process in a simple and engaging way. This project helps users learn how elections work — from voter registration to final vote counting — with beautiful UI and animations.

---

## 🌟 Features

- 📚 Beginner-friendly explanation of elections
- 🎨 Modern and responsive UI
- ✨ Smooth scrolling animations
- 🗳️ Interactive election timeline
- ❓ FAQ accordion section
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
